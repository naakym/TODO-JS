# TODOアプリ (JavaScript)

**Reactとの違いを学習するために、JavaScriptで簡易的なTODOアプリを実装**しました。  
このプロジェクトでは、**React を使用せずに、JavaScript の基本的なDOM操作を用いて実装**しています。  
未完了のタスクを追加し、完了または削除ができます。

## 機能
- TODO の追加  
- 未完了タスクの表示  
- 完了タスクの管理（戻すボタン付き）  
- タスクの削除

## 使用技術
- HTML
- CSS
- JavaScript

## 使い方
1. 「TODOを入力」にタスクを入力する
2. 「追加」ボタンを押すと、未完了リストに追加される
3. 「完了」ボタンでタスクを完了リストへ移動
4. 「戻す」ボタンで未完了リストに戻す
5. 「削除」ボタンでタスクを削除
